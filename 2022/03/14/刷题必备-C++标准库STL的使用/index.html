参考C++标准库STL的使用_xianyuerrr的博客-CSDN博客

公共函数

下述模板类/工具类都有的接口。

clear()

用来清空vector中的所有元素

erase()

  1. 删除单个元素：erase(it)，删除迭代器处的元素；
  2. 删除一个区间内的所有元素：erase(first,last)，即删除(first,last )区间内的元素。

size()

用来获得vector中元素的个数

vector

向量，一种便于拓展的数组超集。

定义

访问

下标访问或通过迭代器访问

常用函数

push_back(i)
pop_back()

删除尾元素

insert(vec.begin()+2, x)

使用迭代器进行数据插入。

note:

类似for循环中，可以使用&i，即通过使用引用来降低程序内存的消耗。

e.g.

for (auto &i: items) {};

set

定义

内部自动排序（增序）且不包含重复元素的容器。如果涉及元素不唯一的情况，需要使用multiset。

类似的unordered_set()，用散列替代set底层的Red Black Tree（一种自平衡二叉查找树），用于处理只去重但不排序的需求，速度快于set。

访问

只能通过迭代器访问

常用函数

insert(x)

将x插入set容器，并自动递增排序和去重，时间复杂度为O(logN)

find(x)

返回set中值为x的迭代器，时间复杂度为(logN)

unordered_set

定义

无序容器，和set的区别就是，unordered_set容器不会自行对存储的数据进行排序。

访问

使用迭代器

C++11新方法

常用函数

empty()

find()

返回迭代器

count()

统计出现的次数

insert()

emplace()

string

定义

访问

下标访问
读入或者输出整个字符串
printf输出string
迭代器访问

常用函数

operator+=
compare operator

两个string可以直接使用==, !=, <, <=, >, >= 比较大小，规则是字典序

length() / size()
insert(), O(N)
  • insert(pos, string)，在pos位置插入字符串 string
  • insert(it, it2, it3)，it为原字符串的欲插入位置，it2和it3是待插入字符串的首尾迭代器
substr(), O(len)

str.substr(pos, len);返回从pos开始、长度为len的子串。

string::npos

是一个常数，本身数值为-1，由于是unsigned_int类型，所以也可以当做unsigned_int的最大值，常作为find函数失配的返回值。

find(), O(nm)

replace(). O(len)

map

定义

映射，可以将任何基本类型映射到任何基本类型。内部是红黑树实现的（set也是），会以键 从小到大的顺序自动排序。

访问

  • 通过下标访问 mp[key]

  • 通过迭代器访问

常用函数

find()

find(key) 返回键为 key 的映射的迭代器，O(logN)

queue

队列先进先出。双端队列deque，优先级队列priority_queue。

定义

访问

由于队列先进先出，所以只能通过 front() 来访问队首元素，或者是通过 back() 来访问队尾元素

常用函数

push()

push(x)，将 x 加入队列，O(1)

front()、back()

front() / back() 获取 队首 / 队尾 元素，O(1)

pop()

pop()，令队首元素出队，O(1)

empty()

empty()，检测队列是否为空，O(1)

注意：使用 front() / pop() 前， 必须使用 empty() 判断队列是否为空

priority_queue

优先级队列，底层是使用堆来进行实现的，队首元素一定是当前队列中优先级最高的

定义

访问

优先级队列没有 front() 函数 和 back() 函数，只能通过 top() 函数来访问队首元素（堆顶）

常用函数

push(x)

令 x 入队，O(logN)

top()

获取队首元素，O(1)

pop()

堆顶出队

empty()

检测队列是否为空

优先级设计

定义

访问

常用函数

获取栈顶元素，O(1)

pop()

empty()

检测是否为空，O(1)

pair

把两个元素绑在一起，但又不想定义结构体时，pair是一个替代品，也就是说pair可以看作一个内部有两个元素的结构体。map的内部实现涉及pair。可以作为map 的键值对来插入。

定义

访问

pair只有两个元素，分别是 first 和 second，按正常结构体去访问就好

常用函数

比较操作数

以<first, second> 元组方式比较

algorithm头文件下常用函数

定义

sort()

排序函数，效率较高。一般不推荐C语言中的qsort()，因为其比较繁琐。

sort规避了经典快排中可能出现的复杂度退化到 O(n ^2) 的极端情况。

STL标准容器中，只有 vector、string、deque是可以使用 sort。因为像是 set、map这种内部是使用红黑树实现的，元素本身就有序。

max() min() abs()

swap()

reverse()

next_permutation()

给出一个序列在全排列中的下一个序列。到达最后一个时返回flase
